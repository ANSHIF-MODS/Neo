# Neo
A simple multifunctional WhatsApp Bot.

## Installation

### Termux

```
> pkg install git
> pkg install nodejs
> pkg update
> pkg upgrade
> git clone https://github.com/AbhiDevOfficial/Neo.git
> cd Neo
> npm install
> node index.js
```
